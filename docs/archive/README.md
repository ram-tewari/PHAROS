# Archived status reports

These files are **historical snapshots** — status updates, "complete" summaries,
phase reports, deployment logs, and one-off assessments written at a moment in
time. Each was accurate for about a day and then drifted.

They are kept for provenance (how the project got here) but are **not
maintained** and frequently contradict each other and the current code. Do not
cite them as the current state of anything.

**For current, maintained documentation see [`notebooklm/`](../../notebooklm/)** —
the six numbered files there are the single source of truth.

Moved here on 2026-07-09 as part of the Phase 1 documentation cleanup.
